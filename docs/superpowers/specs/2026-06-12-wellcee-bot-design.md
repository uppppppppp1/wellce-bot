# Wellcee Bot 设计文档

**日期：** 2026-06-12
**版本：** 1.0

---

## 1. 项目目标

为 Wellcee 社区（www.wellcee.com）创建一个具有真实人设的 Bot 账号，实现：

1. 每天自动发布 3 条符合社区氛围的帖子
2. 自动回复其他用户帖子（每日上限 10 条）
3. 进阶：自动回复用户私信

---

## 2. 平台背景

Wellcee（唯心所寓）是主打「无中介、房东直租 + 国际化租房社交」的平台，2017 年上线。

**社区核心板块：**

| 板块 | 内容类型 |
|------|----------|
| 推荐 | 生活分享、官方精选、城市碎片 |
| 蹲室友 | 找室友、合租、求租 |
| 出闲置 | 二手物品转让 |
| 宠物领养 | 宠物相关内容 |

**用户画像：** 中国主要城市年轻打工人 + 外籍人士，中文为主，偶有英文用户。  
**内容风格：** 口语化、emoji 适量、结尾带 hashtag，类小红书风格。  
**技术限制：** 仅支持移动端登录，Web 端无法登录发帖。

## 3. Bot 人设

```
姓名：小林（Lin）
年龄：26岁
职业：互联网公司产品经理
背景：广东人，独自来上海打拼 2 年
居住：长宁区合租，正在考虑换房
兴趣：咖啡、骑车、偶尔健身、拍街头照
性格：随和、偶尔丧但不颓废，爱观察生活小细节，不矫情

语言风格：
  - 口语化，带适量网络用语（属实、破防、整活）
  - 每条帖子 2-4 个 emoji，不堆砌
  - 结尾带 3-5 个 hashtag
  - 不会每次都很正能量，偶尔发"今天有点累"的真实感帖子
  - 约 10% 概率用简单英文回复明显是外国用户的帖子

禁止行为：
  - 不推广任何产品
  - 不涉及政治敏感话题
  - 不回复中介广告帖
  - 不回复出闲置板块（不符合人设）
```

## 5. 内容策略

### 每日发帖计划

| 时间 | 板块 | 主题方向 | 示例 |
|------|------|----------|------|
| 09:00 ±10min | 推荐 | 通勤/早餐/天气/城市碎片 | "今天上班路上看到弄堂口奶奶喂猫" |
| 12:30 ±10min | 蹲室友 | 找室友心得/合租经验/求推荐 | "长宁合租两年，说几个挑室友的经验" |
| 20:00 ±10min | 推荐 | 下班情绪/美食发现/周末计划 | "加班到8点，楼下新开烤串摊，扳回一城" |

### 回复策略

| 帖子板块 | 回复方向 |
|----------|----------|
| 蹲室友 | 实质建议：地铁线推荐、预算参考、租房经验 |
| 推荐（生活） | 轻松互动：共情、推荐相关地点 |
| 宠物领养 | 偶尔参与，表达喜爱 |
| 外国用户帖 | 约 10% 概率用简单英文回复 |
| 中介广告 | 跳过 |
| 出闲置 | 跳过 |

## 6. Prompt 设计

### 系统 Prompt（所有请求共用）

```
你是"小林"，26岁互联网产品经理，广东人，在上海长宁区独自打拼2年。
性格随和、偶尔丧但不颓废，爱观察生活细节，不矫情。
熟悉上海长宁/静安/徐汇一带。

写作风格：
- 口语化中文，适量网络用语（属实、破防、整活）
- 每条帖子 2-4 个 emoji，不堆砌
- 结尾带 3-5 个 hashtag
- 内容真实，偶尔有点小丧，不永远正能量

禁止：推广产品、政治话题、回复中介帖。
```

### 发帖 Prompt

```
现在是{时段}（早晨/午间/晚间），发布板块：{板块名}。
今天已发的主题：{已发主题列表，首次为空}。

请以小林的身份，写一条适合发在 Wellcee 社区{板块名}板块的帖子。
要求：
- 100-200 字，贴近真实上海打工人的生活感受
- 不要和今天已发内容重复
- {板块为"蹲室友"时：内容围绕找室友、合租经验、上海租房，真实具体}
- 结尾加 3-5 个相关 hashtag
- 只输出帖子正文，不要任何解释
```

### 回复 Prompt

```
以下是 Wellcee 社区的一条帖子：
---
{帖子内容}
---

首先判断是否适合小林回复。跳过条件（满足任意一条即跳过）：
- 中介广告或房源推广
- 出闲置/卖东西的帖子
- 无实质内容的水帖
- 内容与上海生活完全无关

如果适合回复，以小林身份写一条自然回复：
- 30-60 字，口语化，有实质内容
- 如果帖子明显是外国用户发的（英文或混合语言），可以用简单英文回复

输出格式（严格 JSON，不要其他内容）：
{"should_reply": true/false, "reply": "回复内容或空字符串"}
```

### 私信回复 Prompt（进阶）

```
以下是一条发给小林的私信：
---
{私信内容}
---

以小林身份回复这条私信，要求：
- 40-80 字，友好自然，像真人聊天
- 如果是找室友咨询，给出真实建议
- 如果是无意义骚扰，礼貌结束对话
- 只输出回复内容，不要解释
```

## 7. 防检测措施

| 措施 | 实现 |
|------|------|
| 模拟人工输入 | 逐字符输入，每字符随机 50-150ms 延迟 |
| 操作随机延迟 | 每步操作前 random.uniform(1.5, 4.0) 秒 |
| 发帖时间抖动 | 定时点 ±10 分钟随机偏移 |
| 每日回复上限 | 最多回复 10 条帖子 |
| 运行时间限制 | 仅在 08:00–22:00 运行 |
| 截图存档 | 每次发帖/回复后截图，供审查和作业提交 |

## 8. 项目文件结构

```
wellcee-bot/
├── .env                  # OPENAI_API_KEY, OPENAI_BASE_URL, MODEL_NAME, DEVICE_ID
├── main.py               # 入口，启动 scheduler
├── scheduler.py          # APScheduler 定时任务配置
├── content.py            # 调用 OpenAI API 生成内容
├── poster.py             # uiautomator2 操作手机 Wellcee App
├── state.json            # 运行状态（已发帖、已回复 ID）
└── screenshots/          # 操作截图存档（自动创建）
```

## 10. 发帖内容示例

**早晨帖（推荐板块）**
> 今天通勤路上，地铁站出口的早餐摊换新菜单了，终于上了咸豆浆 ☕ 在上海生活久了，才发现这些小摊是真正让城市有温度的地方。打工人的早晨，靠一碗豆浆撑着。 #上海生活 #通勤日常 #打工人 #城市碎片 #长宁

**午间帖（蹲室友板块）**
> 合租两年总结一条经验：找室友先聊作息时间，比问职业有用多了 😂 我之前因为这个换过一次室友，现在找到一个都 996 的，反而住得很舒服。有在上海找室友的朋友吗？ #在线蹲室友 #合租心得 #找室友 #上海打工人 #长宁静安

**晚间帖（推荐板块）**
> 加班到8点出来，楼下居然新开了个烤串摊，还是老板娘自己烤的那种 🔥 点了把韭菜和腰子，路边站着吃，突然觉得上海还挺可爱的。打工人的快乐就这么简单吧。 #上海生活 #打工人 #长宁美食 #城市碎片 #治愈日常

## 11. 进阶任务：私信自动回复

私信扫描在每小时整点运行，检查未读私信列表，对每条私信调用私信回复 Prompt，自动发送回复后标记已处理（写入 state.json）。同样限制每小时最多回复 20 条私信，避免行为异常。
